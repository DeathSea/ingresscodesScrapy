Solution

Whenever you see a lot of text, grouped up in paragraphs and the all number code has a section that never exceeds the number of paragraphs (or lines) in the text, it is generally a good hint at a Book Cipher

Reading each pair of 3 as <paragraph><word><character>, find the characters for each pair
Example:
331 = Paragraph 3, word 3, first letter = Sugar
241 = Paragraph 2, word 4, first letter = In
