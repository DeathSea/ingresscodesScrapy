Code #2

Observations

34 numbers but the range for Ascii is off.

Solution

Let’s make pairs to see what we are working with.

17 37 44 04 64 75 76 45 56 56 86 16 34 64 96 76 77

Looking at the distribution you can see that the second digit of each pairs almost falls in a good decimal to Ascii conversion. Reverse the string

77 67 69 46 43 61 68 65 65 54 67 57 46 40 44 73 71

Almost a good range. Those 40s should be in the 50s range which means we would need to add 1 (or ROT+1) to every digit.
