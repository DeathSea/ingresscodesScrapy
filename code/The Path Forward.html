Three codes could be found hidden in the HTML source of The Path Forward post on the Investigate Ingress blog.

[1] 3obktm6a8etrf62p

[2] m223817133627462635222635174449422327264012

[3] n%663hh^t^46xjp

Code #1

Observations

16 characters with the right amount of numbers for a valid passcode. Same old cipher that has been used in the past.

Solution

Make a 4×4 grid

3 o b k
t m 6 a
8 e t r
f 6 2 p

Reorder columns (3, 2, 4, 1)

b o k 3
6 m a t
t e r 8
2 6 p f

bok36matter826pf

Code #2

Observations

A single letter followed by 42 digits.

Hint
m means 2 things in this code

Solution

M is roman numeral for 1000. In Niantic math it also means minus. We have 42 numbers, make triplets, and from 1000 subtract each numbers

1000- 223 = 777
1000- 817 = 183
1000-  133 = 867
1000-  627 = 373
1000-  462 = 538
1000-  635 = 365
1000-  222 = 778
1000-  635 = 365
1000-  174 = 826
1000-  449 = 551
1000-  422 = 578
1000-  327 = 673
1000-  264 = 736
1000-  012 = 988

Now concatenate all the results together

777183867373538365778365826551578673736988

This is looking more and more like a good Ascii range. Make pairs

77 71 83 86 73 73 53 83 65 77 83 65 82 65 51 57 86 73 73 69 88

Convert decimal to Ascii
