Code #1

Observations

Passcode format is there, go through your usual cipher routine.

Solution

Atbash the letters and numbers

mkb37nearhigh894ky

Near High, remember that usually you also have to find the reverse of the keyword. In this case Near would be Far and High would be Low. Put together it becomes farlow, Farlowe is a valid keyword.

mkb37farlowe894ky

Code #2

Observations

Numbers all separated by a m.

Hint
In this code the M means two things

Solution

Those m could mean a few things, m for minus using Niantic math. m is also 1000 in roman numeral

Lets change those m to 1000
