Code #1

Observations

Exactly the right passcode format so start by the basic ciphers.

Hint
It’s all about physics
Solution

Atbash gives you

kbs94energy942pn

Now energy is not a valid keyword, we need to find the reverse of energy. First thing that comes to mind is mass but it’s not a valid keyword. The next best thing that is a valid keyword is matter.

kbs94matter942pn

Code #2

Observations

Looks almost like the right passcode format but those 8 don’t add up. We can almost see a valid keyword.

Hint
The keyword is exactly what you think now how can you convert that 8 to get the right number or letter

Solution

A bit tricky this one but real fun…

Ok, the keyword has to be Hozho. Now that 8 needs to become a Z. Or is it really a Z we are looking for ? Let’s convert the string to morse

--.- -... ---.. ....- ...-- .... --- ---.. .... --- ..... .---- .---- --- .-

Now reverse that string

-. --- ----. ----. ..... --- .... ..--- --- .... --... -.... ..--- ...- -.--

Convert that back to text

NO995OH2OH762VY

Now it’s reversed but those 8 became 2 which in leet speech become z

Reverse the string and change those 2 to z

yvz67hozho599on

Code #3

Observations

Look like hex but something is wrong with the ranges.

Hint
They used 2 ciphers on this one

Solution

Let’s start by making pairs

86 66 70 37 55 6c 79 76 69 6c 65 63 69 37 52 32 70 68

Take a close look where numbers should be. You have 37 55 and 37 52 32.

In hex numbers range from 30 to 39, in decimal 048 to 57. See how they alternate ?

The cipher alternates between decimal and hex, let’s split them

Decimal: 86 70 55 79 69 65 69 52 70

Hex: 66 37 6c 76 6c 63 37 32 68

Convert to Ascii

VF7OEAE4F
f7lvlc72h

Read down from the upper left V

vff77lovelace742fh
