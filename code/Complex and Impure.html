Observations

Yeah it’s long, real long but… We know that 3 codes are usually released so we have to assume that there are 3 codes is there. Notice some numbers are always repeating. Notice the hint also.

Hint
One code uses the 7s & 8s, the 2 others use the rest of the string and use a ciphers that only need 2 bits.

A lot of sequences of 5 numbers, a few others range from 2 to 4 characters. Again, the hint said 01px, what other cipher uses only 2 bits and has a maximum sequence of 5 characters? It has to be morse but which are – and .?

We know that numbers in morse can use combinations of dots and dashes but they do not alternate (ie: ..—, -…. are valid but .-…, –..- are not). Looking at the first sequence 31221 the only way to get a valid morse number is by using 1&2 to represent a bit and 3&4 to represent the other.
