Three codes were hidden in the HTML source of the NL-PRIME が名古屋へ (NL-PRIME EXPANDS INTO NAGOYA) post on the Investigate Ingress blog.

[1] k7avf9wes7qn63ge

[2] qc4546963gh

[3] 2t5tvqhdgic4o4e3

Code #1

Observations

16 characters with the right amount of numbers for a valid passcode

Solution

Make a 4×4 grid

k 7 a v
f 9 w e
s 7 q n
6 3 g e

Read in a spiral going down from the upper left K

kfs63geneva797qw

Code #2

Observations

Too many numbers but one can be converted to a keyword

Solution

Start by reversing the string

hg3696454cq

Split for passcode format

hg3 69 6 454cq

Convert the 3 to E from leetspeak and 6 to VI from roman numeral

hge69vi4454cq

Code #3

Observations

16 characters with the right amount of numbers for a valid passcode

Solution

Make a 4×4 grid

2 t 5 t
v q h d
g i c 4
o 4 e 3

Read going up from the bottom left O

ogv24iqtech534dt
