Code #1

Observations

The code looks like it’s in the passcode format, so we just need a keyword substitution for c.

Solution

c is pronounced like see in English.

6xbc8seeb6g2t

Code #2

Observations

The length of the code is 15 characters, which has factors of 3 and 5:

4uv
3lk
ktg
9rj
6am

Hint

We see ultra in the middle column but reading vertically down from the top-left has too many numbers in the prefix and no numbers in the suffix.

Solution

Reverse the 2nd and 4th rows:

4uv
kl3
ktg
jr9
6am

4kkj6ultrav3g9m

Code #3

Observations

On first glance, it looks like the distribution of characters are somewhat clustered:

DFGK QRSTWY 3456

Hint

3456 is on the top row of a QWERTY keyboard

QRTWY is on the 2nd row.

DFGKS is on the 3rd row.

Solution

4th row characters are missing. Swapping the first and second rows of the keyboard with the fourth and third rows respectively:

  1234567 <=> ZXCVBNM
QWERTYUIO <=> ASDFGHJKL

threenvbthreerestraintffiveceightw
3nvb3restraintf5c8w
