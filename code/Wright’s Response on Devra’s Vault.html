Three codes could be found in the HTML source of the Wright’s Response on Devra’s Vault post on the Investigate Ingress blog.

[1] dgm55itkxz113pr

[2] mjtwiobzkyuhikj5jqugi2jpmr2csoltmv3gw6q

[3] vrwwoivabdnixiginjv

Code #1

Observations

Passcode format is there, a simple manipulation needs to be done.

Hint
You’ll have to do the same manipulation on the keyword part
Solution

Rot +1 seems to be the way to go

ehn66julya224qs

Now Julya is not a valid keyword or even a character related to Ingress. But… we used Rot+1 what if the keyword was really July a !! After July comes August add that a at the end and you get Augusta

ehn66augusta224qs

Code #2

Observations

39 characters with 5 numbers that are ranging between 2 and 6

Solution

Base32 uses a to z and numbers from 2 to 7. Convert the string from Base32 to Ascii

bgd89V(t)=L(di/dt)9sevkz

V(t)=L(di/dt) is the formula for Inductance which was discovered by Joseph Henry

bdg89henry9sevkz

We are missing two numbers for the suffix sev could convert to 7 but we are still missing a number. Read it as se v, 75

bgd89henry975kz

Code #3

Observations

19 characters, all letters.

Hint
Try to spot the written numbers, it will lead you to all you need

Solution

Recently codes came out with numbers missing the first letters, that wo and iv are a good hint. Let’s split things up in the passcode format

vrw wo iv abdn ix ig in jv

Everything is becoming clearer now. Add the missing for letter to the numbers

vrw two fiv abdn six eig nin jv

We now have

vrw25 abdn 689jv

For the keyword part if we use the logic that some letters are missing and also by just reading it out loud the way it is spelled we find the keyword Abaddon

vrw25abaddon689jv
