Code #1

Observations

Only 3 characters are used <,> and =. One character is used has a split. It then becomes obvious what cipher was used.

Solution
Split at =

>><<< ><<> ><<> <>>> >>>>< < <<>> <<<>> ><<< >><<< <><>

Convert to morse using >=.  <=-
..--- .--. .--. -... ....- - --.. ---.. .--- ..--- -.-.

Convert morse to text
2PPB4TZ8J2C

The keyword part is T which is not a keyword.

By using the logic of the encode:

T in morse is . which is < in the cipher which means LESS than.

Less is a known keyword.
