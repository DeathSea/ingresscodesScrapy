Code #1

Observations

With a length of 31, we can exclude rectangles.
Doing the normal Atbash and Rot steps does not reveal much towards a solution.
But notice something slightly familiar inside the code?

Hint

Xenon lead mercury to gold.

Solution

Identifying that there are elemental symbols in the code, we separate them from each other
Xe Bi Au Hg La Rn Tm Pb Ta Po Ac Hg Xe Ir I Er
Using the Periodic Table, replace each element with its atomic value
54 83 79 80 57 86 69 82 73 84 89 80 54 77 53 68
Convert from decimal to text for the final element.
6SOP9VERITYP6M5D
