Code #1

Observations

Solution

e i 4
6 n u
7 t m

Code #2

Observations

Passcode format is there, go through your usual cipher routine

Solution

Atbash reveals

gyn26noise223xv

Noise is not a valid keyword. There is a lot of keywords that could fit the reverse of noise but signal gave the passcode

gyn26signal233xv

Code #3

Observations

23 characters, mix of lowercase and capital letters

Solution

Mix of capital and lowercase letters could mean Base64 but we have 23 characters which is not good for Base64. Let’s split the lowercase and capital letters

wfplwfhkwznchvp

AEADITOP

Reverse the second string

POTIDAEA

From Wikipedia, Potidaea is a region of Greece, at one point Cassander built a city which was named Cassandreia

Vigenere Autokey the first string with cassandra

ufxtwsetwfifozx

ufx27254zx

Insert the keyword cassandra

ufx27cassandra254zx
