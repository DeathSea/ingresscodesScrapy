Code #3

Observations

All the ingredients are there, usual cipher might apply

Hint
It’s exactly what you are thinking but you will need to play with the columns to get it right

Solution

Make a 3×5 grid

u 3 a
8 q c
n 9 o
3 2 h
w v s

Can you spot the keyword in the 3rd column? Rearrange the rows to get chaos

8 q c
3 2 h
u 3 a
n 9 o
w v s

Still not the right format for a valid passcode. Shift column 1 up by 2 and column 2 up by 1

u 2 c
n 3 h
w 9 a
8 v o
3 q s
