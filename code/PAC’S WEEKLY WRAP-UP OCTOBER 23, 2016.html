Code #1

Observations

All letters ranging between B & G and lots of them repeating.

Solution

With that type of range, letters to numbers is probably a safe bet

If you add up all the digits in the string you get 152. 152 can be divided by 8 which could mean binary. Use Run Length Encoding to get the following binary string

Code #2

Observations

30 letters with no particular range.

Hint
Haven’t seen this in a while but your hint is “A”

Solution

After trying multiple cipher including morse, BiFid, Playfair I opted for one that hasn’t been used in a long time: Vigenere Autokey with A
