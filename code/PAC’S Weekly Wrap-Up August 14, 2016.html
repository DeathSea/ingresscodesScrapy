Code #1

Observations

Right amount of numbers for a passcode, same old cipher.

Solution

Make a 4×4 grid

5 k 3 m
9 x d j
r m a j
x 9 r 8

Read down from the upper right M

mjj83darkxm959rx

Code #2

Observations

20 characters, no particular range for either letters or numbers

Solution

After trying rectangles, morse and a few other cipher, a skip of 2 gave this

ujh34689rvb0117zvzby

You can see the prefix and suffix at the start of the string, let’s split it

ujh34689rv

b0117zvzby

Searching the second string on google gives a reference to the book The Niantic Project: Ingress by Felicia Hajra Lee

A few option for keywords but hajra gets the valid passcode

ujh34hajra689rv

Code #3

Observations

10 characters in what seems to be the passcode format.

Solution

That / gives it away for a keyboard mirror cipher. 1=0, q=p a=; and z=/

zvr42587ww

Now for the keyword, the mirror of / is \ which is glyph for destroy

zvr42destroy587ww
