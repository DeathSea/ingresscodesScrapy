Code #1

Observations

We have numbers, letters ranging from a to j 2 dots and a dash

Solution

There’s a bit of everything in the code ! We know it can’t be hex since letters are ranging from A to J. If you look at the string you can see that every two characters are numbers. Let’s make a skip of 2

78847856535753526869fa.jfgggd-bbh.adidcj

Looks a bit better. The numbers part seems like a good range for decimal to Ascii. Pair them

78 84 78 56 53 57 53 52 68 69

Convert decimal to Ascii

NTN85954DE

We have the prefix and suffix. For the keyword we are left with

fa.jfgggd-bbh.adidcj

Convert letters to numbers using a=0

50.956663-117.038329

These look like GPS coordinates. Search it and google maps will guide you to Mount Syphax

ntn85syphax954de

Code #3

Observations

Symbols, numbers, letters, it’s a mix of just about everything.

Solution
