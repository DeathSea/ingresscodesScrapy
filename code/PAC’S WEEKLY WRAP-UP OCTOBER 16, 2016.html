Code #1

Observations

16 characters with the right amount of numbers for a passcode.

Solution

Make a 4×4 grid

6 i 3 p
t t n 7
e g u c
4 r 7 k

Reverse rows 2 & 4

6 i 3 p
7 n t t
e g u c
k 7 r 4

Read going down from the upper right P

ptc43turing767ek

Code #2

Observations

Passcode format is there, go through your usual cipher routine.

Solution

Using atbash on letters and numbers give

kdx32lawful468tx

The opposite of lawful is chaotic

kdx32chaotic468tx

Code #3

Observations

Numbers and letters, go with the simple stuff at first to get part of the code then you will know exactly what to do.

Solution

While going through the different ciphers here is what Atbash gave

161709iifailurecdh0712

At least we have a keyword. Let’s split things up to make it easier to find the next step

16 17 09 i i failure c d h 07 12

Now everything is clear. Convert letters to numbers and numbers to letters using a=0

qrj88failure237hm

Still invalid. We used atbash, the opposite of failure is success

qrj88success237hm
