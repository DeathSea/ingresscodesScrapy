Code #3

Observations

Looks like all the ingredients are there but it’s missing a number.

Hint
An old favorite but you’ll need to do some manipulation to get it right, try to see if you can find the keyword
Solution

Nothing really obvious to start with, rectangle is usually a good way to start. Looking at the different result, 3×6 is interesting

When there are only 3 columns, the keyword can usually be found in the middle one. If you look closer you can see that it could spell DARSAN if you shift the column up by 4. Darsana is a valid keyword so you’ll need to shift the last column and the first so that you get a valid prefix.

So, shift the first and second columns up by 4 and the third column up by 2
