Code #2

Observations

30 characters. An old favorite will help you

Hint
Veni Vidi Vici

Solution

Looking at the string you can see the word Truth written in reverse. Make a 6×5 grid

a u q k m p
t n i u q t
u q h t u r
t p e s n i
a w e v o n

Read going left from the upper right p

pmkquatquintruthquinseptnovewa

Let’s split things up to make it easier to read

pmk quat quin truth quin sept nove wa

Translate from latin

pmk45verum579wa
