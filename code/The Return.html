Code #1

Observations

You have all the right ingredients. Just jump to the right cipher

Solution

Use a skip of 2 to get

up59467qqjarodisena

anesidorajqq76495pu

anesidora
jqq76495pu

The word anesidora is greek and is related to Pandora

jqq76pandora495pu

Code #2

Observations

Don’t let the extra numbers throw you off…

Solution

Make a 3×5 grid

e o 6
g v 3
v 1 5
8 0 y
9 3 s

Read going down from the upper left E

egv89ov103635ys

OV103 is the Orbiter Vehicle Designation for the space shuttle Discovery

egv89discovery635ys

Code #3

Observations

Something is used to split the code and hint you to the keyword

Solution

Start by reversing the code

pwacipiqbalto5denali7holan9wakinyan9takaani9shantazwanagisyawtom

Searching the words between the numbers gives the Wolf Mountain Sanctuary.
The pack is Balto, Denali, Holan, Shanta, Takaani, Wacipi, Wanagi and a few others.

p wacipi q balto 5 denali 7 holan 9 wakinyan 9 takaani 9 shanta z wanagi s yawto m

Remove the names from the string

pq57999zsm

Reverse

msz99975qp

Keyword is wolfe

msz99wolfe975qp
