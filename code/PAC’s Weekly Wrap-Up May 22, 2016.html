Code #1

Observations

Looking closely you can see that this code contains only 3 bits.

Hint
From “The police” song: I’m sending out an SOS, I’m sending out an SOS…
Solution

Only three distinct values are used, Cha, Law and Neu. Looking at the distribution of Neu, it seems to be a good candidate for splitting.

chachalawlawlaw lawlawlawcha lawlawchalaw lawlawlaw chachalawlawlaw chachalaw chachachalawlaw chalawlaw lawlawchachacha chalawlawcha

If you look at the first sequence we can see 2 x cha and 3 x law which adds up to 5 bits. This is a good start for a morse cipher. Cha=- and Law=.

chachalawlawlaw --...
lawlawlawcha ...-
lawlawchalaw ..-.
lawlawlaw ...
chachalawlawlaw --...
chachalaw --.
chachachalawlaw ---..
chalawlaw -..
lawlawchachacha ..---
chalawlawcha -..-

Morse string:

--... ...- ..-. ... --... --. ---.. -.. ..--- -..-

Convert morse to text

7VFS7G8D2X

Now for the keyword. Searching what Cha Law and Neu meant on Google gave Role Alignment for NetHack.

Alignment is a valid keyword
