Code #1

Observations

Looks like the passcode format but to many numbers.

Solution

Reverse the code

swd5597nztmfh344yu

Atbash the letters and numbers

hdw5513magnus766bf

13magnus is a valid keyword but since it’s Atbash we need to find the reverse of 13magnus which is antimagnus

hdw55antimagnus766bf

Code #2

Observations

17 characters. Right amount of numbers for a valid passcode. A simple manipulation will do it.

Solution

With an odd number of characters sometimes keeping the central letter and reversing the rest does it. Keep the middle n and reverse the first and last group of 8 characters

mts65pronoia929de
