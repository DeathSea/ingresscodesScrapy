Code #2

Observations

22 letters. Try to remember a class that you took in high school to see if you can spot each elements

Solution

Atbash, Rot and morse didn’t give anything. I then realized that some groups of letters seemed familiar. Make pairs

tb ir tm xe te he xe cs ba po tm

These are all chemical elements, get their atomic numbers

65 77 69 54 52 2 54 55 56 84 69

That 2 in the middle is clearly not an Ascii value but convert the rest to Ascii

AME64678TE

For the keyword, he is helium which you could say closely resembles Helios

ame64helios678te
