Three codes could be found in the HTML source of the New Glyphs Appearing? post on the Investigate Ingress blog.

[1] mac92b0000aky3p-9288wk

[2] sxtuqZuwuurwsxturztqrxrwuzututtqsx

[3] lmsgrvrmstrsizvmvhvhdjy

Code #1

Observations

You can see the passcode format, you’ll need to dig deep in your search to get the keyword

Solution

We have the prefix and suffix, let’s split things up to make them more clear

mac92 b0000aky3p-9 288wk

Search b0000aky3p on Google give a DVD on Amazon.

-9 is a hint to chapter 9 of the DVD which is called More. You need to scroll down at the reviews the get the chapters list

mac92more288wk

Code #3

Observations

All letters, go through your usual cipher routine

Solution

Atbash revealed

onhtieinhgihraeneseswqb

Seeing all those E, I and S I looked for written numbers. The code is reversed

bqwsesenearhighnieithno

Let’s splits things to see what we are working with

bqw se se near high ni ei th no

bqw77nearhigh983no

The reverse of near & high would be far & low. Farlowe is a valid keyword

bqw77farlowe983no
