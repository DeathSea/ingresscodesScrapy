Three codes were hidden in the Chatter post on the Investigate Ingress blog.

[1] wsozxnssiifsiiejoifhnw

[2] gsj22xskenyo039ar

[3] zbosymsiubadpabqdkhiqiptsthyil

Code #1

Observations

All letters, bit of weird one but try to find how you can get those numbers

Solution

The string start and end with Ws. Let’s split them

w sozxnssiifsiiejoifhn w

Now for the middle part make pairs

w so zx ns si if si ie jo if hn w

See some numbers now ? You’ll need to skip pairs in the middle part

w zx si si jo hn so ns if ie if w

wzxsisijohnsonsifieifw

wzx66johnson658fw
