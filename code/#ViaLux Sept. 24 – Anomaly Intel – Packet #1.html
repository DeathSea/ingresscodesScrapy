Three codes could be found in the #ViaLux Sept. 24 – Anomaly Intel – Packet #1 post on the Investigate Ingress blog.

[1] ezmurnebhooteeneeewr

[2] 3i9e5nsowtibj9g3

[3] xzu65sfnrorgb464dd

Code #1

Observations

20 letters, you’ll need to look at numbers differently to get it.

Hint
Try to split it into different parts and maybe you’ll see the numbers
Solution

Not an obvious one but split the string up as if you had 2 letter written numbers

ezm ur ne bhoot ee ne ee wr

Usually the numbers are spelled using their first two letters, now they used the last two

ezm FOur NIne bhoot THRee NIne THRee wr

ezm49bhoot393wr

Bhoot is Hindi for ghost.

ezm49ghost393wr

Code #2

Observations

Right amount of numbers for a passcode. Same old cipher

Solution

Make a 4×4 grid

3 i 9 e
5 n s o
w t i b
j 9 g 3

Read down from upper right E

eob39sigint935wj

Code #3

Observations

Passcode format is there, go through your usual cipher routine.

Hint
For the keyword you’ll need to find the extreme reverse of it

Solution

Atbash reveals

caf45humility646ww

We now need to find the reverse of Humility that is also a valid keyword. The antonym of humility is pride. Extreme pride is Hubris.

caf45hubris646ww
