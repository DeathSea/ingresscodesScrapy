Code #1

Observations

42 letters with no particular range. You might want to look at it in a different way.

Hint
Salt would NaCl
Solution

Those ratata are not for Pokemon lovers but are a good hint at the cipher. “at” stands for Astatine with it’s atomic number being 85.

Start by reversing the code

Split the string into pairs

at fr dy ta rn rn ta tm tb bi ac rn ta ta ta ta ra ta rn bi tm

Get the atomic number of each elements

85 87 66 73 86 86 73 69 65 83 89 86 73 73 73 73 88 73 86 83 69

Convert decimal to Ascii

UWBIVVIEASYVIIIIXIVSE

Convert those roman numerals to numbers

UWB46EASY894SE
