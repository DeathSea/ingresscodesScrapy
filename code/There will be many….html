Code #1

Observations

Looks like hex but range is off

Solution

Best thing is to start with pairs

77 78 8b ae a8 7f 9a 7b 7b 96 7d 87 aa aa a9 7b 8e

Note how all number position in the passcode format start with A. Let’s use Hex Atbash

Index:
0123456789abcdef
fedcba9876543210

88 87 74 51 57 80 65 84 84 69 82 78 55 55 56 84 71

Convert decimal to Ascii

XWJ39PATTERN778TG

Code #2

Observations

L00k @t it v3ry c@r3fully

Solution

We have 2 letters at the beginning of the string and 3 at the end. We can assume that the code is reverse

tpq455ucc355346am

Now we have to many numbers. Those 5 and 3 can easily be converted from leetspeak

tpq45success346am
