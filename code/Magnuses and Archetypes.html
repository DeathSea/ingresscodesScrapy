Three codes were hidden in the HTML source of the Magnuses and Archetypes post on the Investigate Ingress blog.

[1] ges48vnkriv114dk

[2] 72apon6tipce2dewzm9

[3] V11JHM0907FTE11OVDUdOUZB

Code #1

Observations

Passcode format is there, go through your usual cipher routine

Solution

Atbash letters and numbers

tvh62empire996wp

The empire is attacked by the rebel…

tvh62rebel996wp

Code #2

Observations

19 characters, once you find the keyword, an unusual split will guide you

Solution

A quick Regex search reveals that Deception is the keyword. Rectangle didn’t give anything but splitting in incremental groups from the middle reveals

72ap on6 ti p ce 2de wzm9

Reverse each groups

pa27 6no it p ec ed2 9mzw

Reverse the whole string

wzm9 2de ce p ti on6 72ap

wzm92deception672ap

Code #3

Observations

Numbers in pairs, capital and lowercase letters, that’s a good hint.

Solution

Convert the numbers to letters and the lowercase letters to numbers using a=0

VlJHMjhFTElOVDU3OUZB

Convert Base64 to Ascii

VRG28ELINT579FA
