Three codes could be found in The Message in the Missions post on the Investigate Ingress blog.

[1] m26mkczz426e

[2] ovw78open675sm

[3] cgl23orev557cz

Code #1

Observations

12 characters, you know you are looking for a two letter keyword

Solution

There are not many 2 letter keywords. Me is one of them that is contained in the string. Simply reverse each group of 6

ckm62me624zz

Code #2

Observations

Looks like a passcode. Open or ∇

Solution

The passcode returns invalid. Open and Accept share the same glyph

ovw78accept675sm

Code #3

Observations

Again, passcode format is there. Try the simple ciphers to get it.

Solution

Atbash the letters and numbers

xto87live553xa

Invalid. The reverse of live is die

xto87die553xa
