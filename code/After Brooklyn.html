Within the post “After Brookly” over at the Investigate Ingress blog, there was a single code found:

jvkrjrjrkdjlddjrjbdkkrkvkikgjidvjrkijr

Code #1

Observations

With a length of 38 characters and multiple repeating characters, this can point to a simple substitution.
Main thing will be to find how to substitute them.

Hint

One step is first needed before the actual substitution can happen.
Do the normal and then look down.

Solution

Trying to work with the text as is bring us no where in finding a substitution index, but we notice something when we used Atbash
qepiqiqipwqowwqiqywppipeprptqrweqiprqi
Look at the code and look at your keyboard where those characters all are (to 80% this is true for everyone, click if you need a QWERTY-US Keyboard)
Notice how everything is in one row and there are numbers right above them?

Substitute each letter with its corresponding number
13081818021922181620080304051423180418
Make pairs
13 08 18 18 02 19 22 18 16 20 08 03 04 05 14 23 18 04 18
and using an alphabet index of a=0..z=25, convert the numbers to letters
NISSCTWSQUIDEFOXSES
Substitute out the numbers for the final code
9SSC2SQUIDE4X7S
