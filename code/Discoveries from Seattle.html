Code #1

Observations

Lots of spaces separated by unicode characters.

Solution

&#8211 is the unicode value for Endash, convert to -. &#8212 is the unicode value for Emdash, convert to — and count the number of spaces between each unicode characters

-7-2-6-5-8-8-5-7-5--5-4-5--5-4-7-8-8-5-

The — should be -0- because there was no space

-7-2-6-5-8-8-5-7-5-0-5-4-5-0-5-4-7-8-8-5-

We can now see a good Ascii range, remove dahses and pair numbers

72 65 88 57 50 54 50 54 78 85

Convert decimal to Ascii

HAX92626NU

To make the code spaces were used, space is a valid keyword

hax92space626nu
