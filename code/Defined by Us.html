Three codes could be found hidden in the Defined by Us post on the Investigate Ingress blog.

[1] cmhorixdznenpsbesf

[2] nfeieoiiitnnenehiiiiifts

[3] bdbowiozexkf

Code #1, #2, & #3

Observations

Yes they all work together except that it’s either a bad paste from Niantic or a new way to make codes harder to get !

Solution

With all those letters in the second string we can guess they make up most of the numbers. Let’s group the 3 strings together

cmhorixdznenpsbesfnfeieoiiitnnenehiiiiiftsbdbowiozexkf

Make a 6×9 grid

c m h o r i
x d z n e n
p s b e s f
n f e i e o
i i i t n n
e n e h i i
i i i f t s
b d b o w i
o z e x k f

Reading down you can see a good start to the Boson keyword. The rest seems to be in the 4th column but the S is missing. Seems like we have a missing row but start by rearranging columns as 1, 4, 2, 5, 3, 6

c o m r h i
x n d e z n
p e s s b f
n i f e e o
i t i n i n
e h n i e i
i f i t i s
b o d w b i
o x z k e f

If you read down you get

cxpnieibo?oneithfox?mdsfinidz?resenitwk?hzbeieibe?infonisif?

Let’s split things up

cxpnieibo?oneithfox?
mdsfinidz?resenitwk?
hzbeieibe?infonisif?

The “?” represent the missing string. For the keyword parts we can guess the keywords as Boson, Dzire or Desire and Begin

cxpnieibosoneithfox?
mdsfinidziresenitwk?
hzbeieibeginfonisif?

Which gives

cxp98boson834x?
mds59desire792k?
hzb88begin496f?

You can now try every letter to guess the missing one for each code.

cxp98boson834xt
mds59desire792kx
hzb88begin496ff

xco98boson834xt is invalid, substitute boson with higgs

cxp98higgs834xt
mds59desire792kx
hzb88begin496ff
