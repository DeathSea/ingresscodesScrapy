Code #3

Observations

Almost looks like the passcode format but you’ll need to manipulate it. The keyword sounds like…

Solution

With only two letters at the beginning and end of the passcode, it might indicate that somethings are reversed. There is a total of 14 characters so let’s reverse each group of 7

nhu87abdn537v

We now have a valid prefix and suffix. For the keyword, if you say it out loud real fast it sounds like Abaddon
