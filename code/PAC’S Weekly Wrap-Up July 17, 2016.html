Code #3

Observations

All number, check to see if you can find a pattern.

Hint
Check for the passcode format to see what you need to modify. That -1 at the end is also a huge hint

Solution

Looking at where numbers should fall in the passcode format we see that they are all in the the 60s range. It could be assumed that you need to remove 1 for every number in every pairs but then that -1 doesn’t work. The only way to make that one work is by adding 1 so it becomes zero so we need to alternate between subtracting 1 and adding 1.
