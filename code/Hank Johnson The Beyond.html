Three codes could be found in the Hank Johnson: The Beyond post on the Investigate Ingress blog.

[1] ywHMmZjVyNnJDxjLzdq2m25X

[2] wvt37gzxgrxzo571xx

[3] 4240466460495639444243326566693042

Code #1

Observations

24 characters with a mix of numbers, capital and lowercase letters, big hint on the cipher to use.

Solution

24 characters with that pattern has got to be Base64. Inverse the case of the letters

YWhmMzJvYnNjdXJlZDQ2M25x

Convert Base64 to Ascii

ahf32obscured463nq

Code #2

Observations

Passcode format is there, go through your usual cipher routine

Solution

use Atbash on letters and numbers

deg73tactical539cc

The reverse of tactical is strategic

deg73strategic539cc

Code #3

Observations

All numbers but not in the right range for Ascii

Solution

Let’s start by making pairs to see if we have a pattern

42 40 46 64 60 49 56 39 44 42 43 32 65 66 69 30 42

Looking at where numbers should fall in the passcode format, the numbers all start with 6. We need to make 6=5. The only way to do that is to Atbash the numbers using 0=1, 2=9…

79 71 75 57 51 72 65 82 77 79 78 89 56 55 52 81 79

Convert decimal to Ascii

OGK93HARMONY874QO
