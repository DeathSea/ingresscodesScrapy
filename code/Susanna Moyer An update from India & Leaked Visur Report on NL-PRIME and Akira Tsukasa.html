Code #2

Observations

All letters, lots of them repeating.

Hint
F, S, E, N, O, I how can you make those numbers
Solution

With that amount of S, F and I it could probably be 2 letter numbers (SI, SE, EI, NI…)

Make 2 rows

SEESSFTFFFNSFESSFSSFOFTSZNES
IIIEEWINIIEIEIIIIIIOIWIEIIII

You can now see a few numbers spelled out but some like FW is not good. Reverse the bottom row

SEESSFTFFFNSFESSFSSFOFTSZNES
IIIIEIWIOIIIIIIEIEIINIWEEIII

Here’s the number sequence it gives

6886752545965867576515270986

Make pairs

68 86 75 25 45 96 58 67 57 65 15 27 09 86

Looks like decimal to Ascii but some numbers are off. Notice where the 5 are which should be numbers, we need to reverse the string first

68 90 72 51 56 75 76 85 69 54 52 57 68 86

Convert decimal to Ascii

DZH38KLUE649DV

Code #3

Observations

All the ingredients are there, if you are thinking that same old cipher that has been used lately you are partly right.

Hint
You’ll need to find the keyword first to help you organize everything
Solution

Let’s start by making a 4×4 grid

i 4 d 2
3 s h o
e s v v
8 r 7 l

Not much of a code here even by reversing rows. Let’s try to see if we can spot the keyword. We know there are 16 characters so we are looking for a 6 letter keyword. Browsing through the list Helios seems to be the keyword. Let’s see if there is a way of shifting those rows to get the right passcode format.

Shift rows to the left, row 1 by 2, row 2 by 1, row 3 by 3 and row 4 by 2 to get

d 2 i 4
s h o 3
v e s v
7 l 8 r

Read down from the upper left D

dsv74helios843vr
