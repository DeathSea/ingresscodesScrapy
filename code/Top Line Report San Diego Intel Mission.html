Code #1

Observations

We see a set of limited alpha characters.

Hint
It may be easier to dial with numbers
Solution

In pairs we notice the first digit is always low (1-3), the second is 2-9.

33 36 38 38 36 18 19 13 23 26 34 12 35 25 26 34 14 18 19 28

This resembles a phone cipher, similar to (non-predictive) T9 texting.  Find the 2-9 digit and select the 1-4 letter.  eg: 3rd letter on the “3” button

FOVVOTWDENIALKNIGTWU

Replace the numbers (first two letters) with numbers

4VVO2DENIALK9G2U

Code #3

Observations

This code is nested withing <SPAN> tags which may offer some hints:

<span style="font-size: 1px; letter-spacing: -1px;">6983098964768885779162966598

Hint
The numbers looks close to decimal range.  Perhaps we should apply the SPAN hints (-1, +1)
Solution

Using the span hints we ROT alternating digits -1, +1

-1: 5 7 9 7 5 6 7 7 6 8 5 8 5 8
00: 6983098964768885779162966598
+1:  0 4 0 0 5 7 9 6 8 2 3 7 6 9

Combine the pairs

 50 74 90 70 55 67 79 76 68 82 53 87 56 89

Convert from Decimal

2JZF7COLDR5W8Y
