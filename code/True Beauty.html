Code #1

Observations

All letters, try to spot some numbers in there to help you solve it

Solution

Looking closely you can see some numbers written in reverse (is es es) which would convert to 677. Start by reversing the string

hbboneisvovmfhsesesikn

Split it into the passcode format

hbb on ei svovmfh se se si kn

We now have

hbb18svovmfh776kn

Atbash the string including the numbers

syy92helenus334pm

Helenus is the twin brother of Cassandra in greek mythologie

syy92cassandra334pm
