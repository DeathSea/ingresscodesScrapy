Three codes appeared hidden throughout the Lightman’s Recovery post on Investigate.Ingress

[1] 2UNS9ENORDS7A4H
[2] yowftgyowlehb;sbbebinlimiby
[3] aammmkxxxammxammxmaamkxammxmammmxxxammxmammkxxxammmkxxxaxxammakammmx

Code #1

Observations

The code is already in passcode format except for the keyword

Solution

Take the code as 3 groups of 5:
2UNS9 ENORD S7A4H
Notice that ENORD reversed is DRONE, a valid keyword. Reverse the groups but not the order:
9SNU2DRONEH4A7s

Code #2

Observations

All letters with the exception of ‘;’. YOW appears twice at the start, separated by 3 characters.

Solution

QWERTY keyboard, reverse the rows.
twojyhtwosignalnninebsevent
2jyh2signaln9b7t

Code #3

Observations

Only 4 unique characters in the code, all letters and letter distribution not uniform.

Hint

Maybe Julius can help you.

Solution

Apply Rot11 (Ls capitalized to make it easier to read):
LLxxxviiiLxxiLxxixLLxviLxxixLxxxiiiLxxixLxxviiiLxxxviiiLiiLxxLvLxxxi
Split at Ls:
L Lxxxviii Lxxi Lxxix L Lxvi Lxxix Lxxxiii Lxxix Lxxviii Lxxxviii Lii Lxx Lv Lxxxi
Roman numerals to decimal numbers:
50 88 71 79 50 66 79 83 79 78 88 52 70 55 81
Decimal to ASCII:
2XGO2BOSONX4F7Q
