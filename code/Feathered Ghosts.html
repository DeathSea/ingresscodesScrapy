Code #1

Observations

54 characters, look like hex but not in the right range.

Solution

Let’s try hex atbash. So your index is:

0123456789abcdef
fedcba9876543210

04b0db0cc06c0ac0ca0ca0ab0da08b00b0da0fa06c0ac0aa02b04b
fb4f24f33f93f53f35f35f54f25f74ff4f25f05f93f53f55fd4fb4

Notice how “f” seems to be used as a split ? Reverse the string to see it better

4bf4df55f35f39f50f52f4ff47f52f45f53f53f35f39f33f42f4bf

Make groups of 3

4bf 4df 55f 35f 39f 50f 52f 4ff 47f 52f 45f 53f 53f 35f 39f 33f 42f 4bf

Remove the trailing F of each triplets

4b 4d 55 35 39 50 52 4f 47 52 45 53 53 35 39 33 42 4b

Convert Hex to Ascii

KMU59PROGRESS593BK

Code #2

Observations

A mix of just about anything, you’ll need to look at it in a different way.

Hint

After playing with it a bit I noticed something when reversing the string

See the keyword presquevu in there ? It’s a mix of roman numeral and leet speach
