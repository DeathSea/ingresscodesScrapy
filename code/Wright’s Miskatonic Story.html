Code #3

Observations

60 digits ranging from 1 to 5

Hint
You might think of something easy with that number range but you’ll need to feel it to get it

Solution

Numbers ranging from 1 to 5. Could be a Polybius square but after decoding it doesn’t give much. Groups of 2 or 3 don’t give any good pattern. One interesting thing is that the sum of all the digits is 108. Not binary but it can be divided by 6. Let’s use Run Length Encoding using dashes and dots
