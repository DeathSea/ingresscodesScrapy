Code #3

Observations

Thanks to punch neko & Riiingo for submitting the solve for this code!

We have all letters, but the numbers 0 and 1 are present.

Hint

You’ll need an old school type of phone to send this code to friends.

Solution

Using the Mobile phone keypad layout (also called Vanity code), replace each letter with the number that it is found on, keeping the 0’s and 1’s as is. But instead of the normal method where you repeat the number for how many times you need to press that key to gain the letter, we just use a single number per letter.
90797854537377806982706967845351508781
Pair them up
90 79 78 54 53 73 77 80 69 82 70 69 67 84 53 51 50 87 81
and convert from decimal to text
ZON65IMPERFECT532WQ
