Code #1

Observations

Right amount of numbers for a passcode. Same old cipher that’s been used over and over again.

Solution

Make a 4×4 grid

2 n 6 s
9 a h z
t h a e
j 9 n 9

Read down from upper right S

sze96hannah929tj

Invalid. Hannah is Misty Hannah.

Code #2

Observations

60 characters all made of / & \

Hint
You’ll need to feel the code to get it

Solution

There are 60 characters, divided by 6 it would make 10 characters which is what we need to get a prefix and suffix. 6 bits for each characters make it a good candidate for braille.

Start by making a 3×20 grid

Use a transpose cipher to rotate it

//////\\\\\\\\\\//\/
/\\/\\///////\/\\///
\\/\/\///\///\//\\\\

Split for Braille

// // // \\ \\ \\ \\ \\ // \/
/\ \/ \\ // // // /\ /\ \/ //
\\ /\ /\ // /\ // /\ // \\ \\

/ are dots and \ are blank, convert braille to text

fnm76728dj

Now for the keyword, the code use /\ which is glyph for attack

fnm76attack728dj
