Three codes are hidden in the alt texts of the OLW’s Global Portal Survey post on Investigate.Ingress

[1] ie.oqyg,wbyu;,yqieplei
[2] xevifwdsxisurebsczeninothgierq
[3] uavavhd7ksex34p2

Code #1

Observations

Letters mixed with .,; hint to the keyboard layout being involved.
Repetition of ‘ie’ in the proper places of the format, if digits are abbreviated by 2 letters.

Solution

Replace each letter with the one on the ‘opposite side’ of the row on a QWERTY keyboard.
(So q swaps with p, w with o, a with l, et cetera.)
eixwpthcontractpeiqsie
Replace abbreviations by digits in the proper places:
8xpw3contractp8q6e

Code #2

Observations

We can make out several digits written backwards.

qr8o9zcsberu6sdw5x

8rqz9obscureds6x5w

Code #3

Observations

4 digits and mostly common letters indicate a simple rearrangement.

Solution
