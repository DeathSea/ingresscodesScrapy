Code #1

Observations

Passcode format, numbers are off since 0 is not used in daily code format. That hint in the HTML is very important.

Solution

That font-size: 2px hint usually means to either make a 2 by x grid or could also mean Rot+2 which would bring those numbers to a valid format. Let’s try that.

6far7ccu4m2g

We now need a substitution for CC. Knowing the different characters in the Ingress story, CC could very well be Carrie Campbell. Both are valid keywords, Carrie gave the valid passcode.

Code #2 & 3

Observations

All letters, applying different method didn’t give a lot until noticing that 2 & 3 have something in common.

Hint
#3 is the first step for #2

Solution

After trying Atbash, ROT, Vigenere, Transposition, Morse, Playfair and a few other cipher to code #2 I got nothing. I then noticed that the Atbash of code #2 gave code #3. Can’t be coincidence.

First step: Atbash

Skip of 2

nizozthkthnvveldniwdgnirps

It’s hard to see at first glance but there is a code is there. Let’s split it to make it easier

ni zoz th k th n v v eldniwd gnirps

Let’s changes those to numbers, V is roman numeral for 5 and let’s reverse that last part for the keyword
