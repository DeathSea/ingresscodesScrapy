Code #1

Observations

Capital letters, lowercase and numbers. The equal sign gives it away for the first step.

Solution

Clearly this is Base64, convert to text

Z!B!O!X$K!H#L)X^R&S!X%M!Q&I!X$M!Q*I!W%M)B$L!G&V

We now have alternating letters and symbols. Use a skip of 2 to split them

ZBOXKHLXRSXMQIXMQIWMBLGV!!!$!#)^&!%!&!$!*!%)$!&

ZBOXKHLXRSXMQIXMQIWMBLGV
!!!$!#)^&!%!&!$!*!%)$!&

Convert the symbols to their respective numbers on the keyboard

11141306715171418150417

We now have 23 numbers. Range for Ascii is not valid. Let’s try to split to see if we can get a range for numbers to letters

11 14 13 06 7 15 17 14 18 15 04 17

Ignore that 7 for now and convert the rest to letters using a=0

LONG7PROSPER

Convert that 7 back to symbol

LONG&PROSPER

The word missing from the quote is live

Vigenere autokey the string of letters with the passphrase live

Code #2

Observations

Passcode format is there

Hint
Don’t be misled by the keyword, this is not a star wars theme

Solution

Atbash letters and numbers gives

yur39jedi425mf

That jedi keyword was probably there to mislead people with all the star wars hype going on. You need to switch the first three letters (yur) with the first three letters of the keyword (jed)

jed39yuri425mf

Code #3

Observations

15 characters with the right amount of numbers for a valid passcode.

Solution

A regex search reveal that boson should be the keyword. Trying to use rectangle to get everything in order doesn’t work. Let’s split in groups of 1, 2 , 3, 4 and 5

y qj b34 noso nj375

Reverse each group

y jq 43b oson 573jn

yjq43boson573jn
