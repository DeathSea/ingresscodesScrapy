Solution

cwn45acolyte496ad

Code #2

Observations

16 characters with enough numbers for a valid passcode. This time with a different twist !!

Solution

Make a 4×4 grid

c i v 7
e g 8 u
r q 4 x
n v i q

Read in a spiral starting at the lower right Q

qxu7vicernvi48gq

Convert the roman numerals to numbers

qxu76cern648gq
