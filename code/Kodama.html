Passcode #2

Hint
You used the symbols for passcode #1. Take a good look at the different kind of letters

Solution

We used the symbols for passcode #1, that leaves the letters. There are upper and lowercase letters. For this passcode only take only the uppercase letters

Passcode #3

Hint
We used symbols and uppercase letters for the other codes. The lowercase letters will help you for something but you’ll need the full string and a few steps for the rest

Solution

We used symbols and uppercase letters for the other codes. Let’s get the lowercase letters

You can see some reversed words in there, split them.

Reverse each word

imagination is more important than knowledge knowledge is limited imagination encercles the world

This is a quote from Einstein. We now have a possible keyword.
