There were 3 codes found on the Flash Shard post on the Investigation Blog.

[1] r9h6jneiw9gsf5
[2] 4fgb4hgilmtb6f1g
[3] 8i75rlsb2spsjbos

Code #1

Observations

The code looks like the passcode format, but backwards:

5fsg9wienj6h9r

However, wien isn’t a keyword.

Hint

Wien is the German name for Vienna. Perhaps there’s something special happening there.

Solution

The Acolyte made an appearance in Vienna for Obsidian. acolyte is a keyword.

5fsg9acoyltej6h9r

Code #2

Observations

The code is in the normal passcode format but there is a 1 in the code.

Solution

Applying atbash to the code:

6uty6strongy4u9t

The code is still invalid, there is one more step. The opposite of strong is weak:

6uty6weaky4u9t

Code #3

The code has a string length of 16, which can be in a 4×4 transposition:

8i75
rlsb
2sps
jbos

Solution

Reading top-down from the top-right corner:

5bss7spoilsb8r2j

The code is still invalid. There is a phrase: to the victor goes the spoils, victor is a keyword:

5bss7victorb8r2j

The code is still invalid! Victor Kureze is a character in the Ingress world.

5bss7kurezeb8r2j
