Code #1

Observations

The code has 25 letters in it, which has multiples of 5 and 5:

ycegu

rwpei
htmhf
tqoxw

Solution

Reading upwards from the bottom-left:

thruyqtwocomplexheigwfivu
3uyq2complexh8w5u

Code #2

Observations

The code has 18 characters (14 letters, 4 numbers), it is a perfect candidate to attempt a transposition:

7uedpn
d6ary9
hddo3m

Solution

Reading downwards from the top-left:

7dhu6deaddropy3n9m

Code #3

Observations

The code looks like it is in the passcode format but there is a 1 which is not allowed in that format.

Solution

Using atbash on both the numbers and letters:

8otc9liberateu4o4e
