Code #1

Observations

15 characters with the right amount of numbers for a passcode.

Solution

Make a 3×5 grid

o q 3
k u 2
o a 3
4 r q
8 k a

Read down from upper left O

oko48quark323qa

Code #2

Observations

16 characters, again with the right amount of numbers for a passcode.

Solution

Make a 4×4 grid

7 r 7 p
c u e c
h k z 8
g 4 e 8

Read up from bottom left G

ghc74kureze788cp

Invalid. Kureze is Victor Kureze.

ghc74victor788cp
