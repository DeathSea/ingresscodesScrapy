Code #3

Observations

Here’s an interesting one. Those \ have to mean something. Try counting the numbers of letters to see what you get.

Solution

Counting the letters you get 24. That would lead us to Base64. Now that the new passcode format doesn’t start with a number it’s probably harder to spot Base64. Start by changing the case of all character except those right after the \

hN3N3YzchlmbhVmcnNjNkdme

Converting this from Base64 to text doesn’t give anything good. Let’s try reversing it
