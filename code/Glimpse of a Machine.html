Three codes were found hidden in the Glimpse of a Machine post on the Investigate Ingress blog.

[1] 0oj78ymar039xx

[2] or2668k757ksv

[3] RiwiiNsIbTaUlTpUaRiBtNsEtUlwOp

Code #1

Observations

Aside from the leading 0, looks like a passcode format.

Hint
You’ll need a dash of manipulation and a dot of roman salad
Solution

With those 0 first instinct was to try Atbash but that didn’t fix the keyword. Let’s try morse

----- --- .--- --... ---.. -.-- -- .- .-. ----- ...-- ----. -..- -..-

Swap morse

..... ... -... ..--- ...-- .-.. .. -. -.- ..... ---.. ....- .--. .--.

Convert morse to text

5SB23LINK584PP

5 can either be S in leetspeak or V in roman numeral. The valid passcode is

VSB23LINK584PP

Code #2

Observations

Good passcode format if you look at it in a different way. Just needs a fix in the middle part.

Solution

Reverse the string

vsk757k8662ro

Searching 7k8 on Google gives the airport code for the Martin Field airport

vsk75martin662ro

Code #3

Observations

Mix of capital and lowercase letters…

Hint
This is a Periodic cipher

Solution

Lowercase and capital letters, almost split up in pairs, this looks like a Periodic Table element cipher. You can see Ua, Tp which reverse become aU and pT for gold and platinum. Reverse the string

pOwlUtEsNtBiRaUpTlUaTbIsNiiwiR

Split before lower case letters

pO w lU tE sN tB iR aU pT lU aT bI sN i i w iR

These are all chemical elements, get their atomic numbers

84 74 71 52 50 65 77 79 78 71 85 83 50 53 53 74 77

Convert decimal to Ascii

TJG42AMONGUS255JM
