Code #3

Observations

You can sort of see the passcode format in there and a keyword.

Solution

Reverse the string

fdf7.63soul8.512v.25

We need to fix those numbers. Those dots are really divide signs

63/7=9 and 512/8=64

fdf79soul864v.25

Interpret the V as roman numeral 5. You will have

5.25 = 25/5 = 5

fdf79soul864 5 5

Convert those 5 back to roman numerals

fdf79soul864vv
