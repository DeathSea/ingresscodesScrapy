Three codes were hidden in the HTLM source of the Sixteen Twelve Ten post on the Investigate Ingress blog.

[1] x0i2j22614186151l2g3

[2] 6i4a3adcplevq9n9

[3] gfa61vhkkzqc285or

Code #1

Observations

If you count the letters you have just enough for a prefix and suffix. How you modify them is the tricky part.

Solution

Using a skip of 2 will give you somewhat of a passcode format

xij21165lg0226481123

Split the string in two

xij21165lg
0226481123

Search the second string on Google to get a reference to a book by Bernard Lightman. We now have the keyword but the ‘fixes needs so work.

To get it you need to Rot the first string minus the second to get

xgh67354jd

Insert the keyword

xgh67lightman354jd

Code #2

Observations

16 characters with the right amount of numbers for a passcode

Solution

Make a 4×4 grid

6 i 4 a
3 a d c
p l e v
q 9 n 9

Read going down from the upper right A

acv94denial963pq

Code #3

Observations

Passcode format is there, go through your usual cipher routine to find the hint to the keyword

Solution

Rot +1 reveals

hgb72willard396ps

Willard is a movie where Willard unleashes an attack led by an intelligent rat called ben

hgb72ben396ps
