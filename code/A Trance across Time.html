Code #1

Observations

Passcode format is there, go through your usual cipher routine.

Solution

Atbash the letters and numbers to get

qcp98retreat927rx

Invalid? Remember that you need to find the reverse for the keyword. Reverse of retreat is advance.

qcp98advance927rx

Code #2

Observations

30 characters without any specific range.

Hint
If your first instinct is to use -. you are right. You’ll need a first step before using that.

Solution

Converting to morse give a good range of 120 characters

.- -..- -..- --.. --.- --.- --.. --.. --.. --.. --.. ---.. ---.. .--. --- --.. .-. -.-. --.- ---.. ..-. .--- -.-. .--. ..-. .---- .--- .--. --.- .--.

Groups of 8 gives

.--..--. .---..-- .---.--- ..--..-- ..--..-- ..---..- --...--. -----... -.-.-.-- .----... .-..---- .-..--.. .-..---- .---.--. --.-.--.

If .=0 and -=1 the start is good for binary to Ascii but then the rest doesn’t add up.

Let’s try reversing the original string first

pqpj1fpcjf8qcrzop88zzzzzqqzxxa

Convert this to morse

.--. --.- .--. .--- .---- ..-. .--. -.-. .--- ..-. ---.. --.- -.-. .-. --.. --- .--. ---.. ---.. --.. --.. --.. --.. --.. --.- --.- --.. -..- -..- .-

Make groups of 8

.--.--.- .--..--- .----..- ..--.-.- ..---..- .---..-- .--.-..- .--..--- .--.---. .---..-- ..--..-- ..--..-- ..--.--- .---..-. .--..-.-

We now have a perfect string to convert to binary. dots=0 dashes=1

01101101 01100111 01111001 00110101 00111001 01110011 01101001 01100111 01101110 01110011 00110011 00110011 00110111 01110010 01100101

Convert binary to Ascii

Code #3

Observations

44 numbers, try to find what numbers appear often to help you find the cipher.

Hint
You might want to phone a friend

Solution

Let’s start by making pairs to see what we have

52 63 94 32 43 32 43 32 92 71 53 63 73 32 81 42 33 63 32 43 51 72

Not a good range for hex or decimal but notice how the last number of every pairs is between 1 & 4? That could indicate a phone cipher

Reverse the string of pairs

27 15 34 23 36 33 24 18 23 37 36 35 17 29 23 34 23 34 23 49 36 25

Using the phone cipher convert everything to letters (ex: 27 is the second letter of 7 on a phone which is q)

You should get

qjieofhterolpxeieiezok

Reverse

kozeieiexplorethfoeijq

koz88explore348jq
