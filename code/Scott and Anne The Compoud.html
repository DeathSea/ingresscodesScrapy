Code #1

Solution

Code #3

Observations

All letters, again go through your normal routine

Solution

With all those letters you can see some numbers can be spelled out. That will usually indicate a rectangle cipher

Make a 7×3 grid

t e i t d t t
w w e s o j h
r s u i c r m

Reverse the second row

t e i t d t t
h j o s e w w
r s u i c r m

Read down from t
