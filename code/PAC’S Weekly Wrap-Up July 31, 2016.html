Three codes could be found in the HTML code of the PAC’S WEEKLY WRAP-UP: JULY 31, 2016 post on the Investigate Ingress blog.

[1]  nqkadvancechaoscreatedatajourneycx

[2] seuviiixvixiiivga

[3] uobhteenineeeeeqr

Code #1

Observations

Look closely and try to find how each glyph could convert to numbers

Solution

Let’s start by splitting everything up

nqk advance chaos create data journey cx

We have the 3 letters plus 2 glyphs for the prefix then 3 glyphs and 2 letters for the suffix. Need to figure out how to convert those glyphs to numbers. Look at how many dots each glyph crosses on the calibration grid.

Advance is 3, Chaos is 8, Create is 5, Data is 5 and journey is 7

We now have the prefix and suffix

nqk38 557cxà

The middle part used a glyph to convert to a character, glyph is a valid keyword

nqk38glyph557cx

Code #2

Observations

A mix of letters and what seems to be roman numerals.

Solution

There are multiple ways you could split the code because of the roman numerals but here’s how it should be split

seu vii ix v ix ii iv ga

Convert the roman numerals to numbers except for the middle v which will become the keyword.

seu79v924ga

V can represent a few glyph, after a few tries Defend is the right one

seu79defend924ga
