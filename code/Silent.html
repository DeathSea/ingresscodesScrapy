Code #1

Observations

You can spot a few written numbers in there. A bit of a rearrange will give you the code.

Solution

o f

m x

Read from the lower left M

mxefofoaframfosesigp

mxe44afram476gp

Code #2

Observations

Right amount of numbers in there. Overused cipher applies

Solution

Make a 4×4 grid

4 e 8 j
7 v g y
p a e e
y 6 n 9

Read down from the upper right J

jye98geneva647py

Code #3

Observations

All numbers, you can spot the range and know what you have to do.

Solution

Start by reversing the string and splitting in pairs

70 85 56 52 55 67 73 84 78 65 73 78 52 54 81 77 79

Convert decimal to Ascii

FU847CITNAIN46QMO

Reverse

OMQ64NIANTIC748UF
