Three codes appeared hidden throughout the Portal Clue: Obsidian Journey post on Investigate.Ingress.

[1] 5ivm5xrerorazgrlmq5b8y
[2] 0dri5uxzbgx7q2r
[3] fyzcopxrxc4qrbypzc347xqw

Code #1

Observations

Code is in passcode format but keyword is random letters and i is an invalid letter for media codes.

Solution

Atbash, numbers included.
5ren5civilizationj5y2b
Civilization glyph reflection of Nature glyph.
5ren5naturej5y8b

Code #2

Observations

Code is in passcode format but keyword is random letters and 0 is an invalid number for media codes.

Solution

Rot 7.
7kyp2begine4x9y

Code #3

Observations

The code has an unusually long morse sequence.

Solution

Convert to morse and remove spaces.
..-.-.----..-.-.---.--.-..-.-.-..--.-.....---.-.-.-...-.--.--.--..-.-....--....---...-..---.-.--
Split into 3 columns and transpose to get 3 rows.
..---.--..---.-.-.-----....-..-.
.--..-...-.-..--........-.-.--.-
-.-.-----.....-.-.---.-.-.-..---
Braille using . as blank and – as dot.
9pzx9darkxmk2s4z
