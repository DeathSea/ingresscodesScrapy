Code #3

Observations

A mix of alternating letters and symbols.

Solution

Use a skip of 2 to separate the letters and symbols

wvhtofofififpoq@*>@@^@>%#>$($(

Using only the letters, reverse them

qopfififofothvw

We now have the prefix and suffix

qop55 443vw

“Un-shift” the symbols
