Code #3

Observations

All letters, you’ll need to rearrange and look at it differently to get the first step.

Hint
One will lead you to the key to unlock the second

Solution

Atbash, Rot, morse didn’t give anything. Then skip of 2 gave something the caught my eye

roopmotlliwerujnignihtonhzdqihfwlifuwiitvquarax

Look at the start of the string, seems like the words are reversed

Let’s split the string in two

roopmotlliwerujnignihton

hzdqihfwlifuwiitvquarax

Reverse each words of the first string

poor tom will injure nothing

Search on Google gives the name of a poem called Tom o’Bedlam

Vigenere the second string with bedlam

gvafivesixfivefivetwopx

gva56bedlam552px
