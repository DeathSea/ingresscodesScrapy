Three codes could be found hidden in the Hank Johnson: Reawaken ADA at #ViaLux post on the Investigate Ingress blog.

[1] 5zg0nm3622j9a0h10n

[2] 97w6237363379647v616w616x6365323z61727

[3] ecph45ke264pyz

Code #1

Observations

Mix of letters and numbers.

Hint
Take a good look at the numbers, maybe some are not numbers!
Solution

There are 18 characters so we are looking for an 8 letter keyword. There’s a lot of noise in this code with all the numbers but start by making a 2×9 grid

5 z
g 0
n m
3 6
2 2
j 9
a 0
h 1
o n

Now reverse every second rows

5 z
g 0
n m
3 6
2 2
j 9
a 0
h 1
o n

Read down from the upper right Z

zgm32j0hn50n629a1o

Convert from leet speak

zgm32johnson629aio

That io at the end should be 10 and convert to x in roman numerals

zgm32johnson629ax

rqj25clamantis672my

Code #3

Observations

The passcode format is sort of there.

Solution

Not sure how to explain this one but here’s how we got it.

Make a 2×14 grid

e c p h 4 5 k
e 2 6 4 p y z

You can see the prefix and suffix in the middle of each rows.

cph45 264py

You are left with eekz which can make up zeke for ezekiel calvin
