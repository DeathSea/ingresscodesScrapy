Three codes were found in the annotations of the Ingress Report of June 16th.

[1] .--. ---. --.- ..--- ----. .. -- .- .-- --... ----. ..--- .--. ..--

[2] te259beol57mbr

[3] ni9gq92to9

Code #1

Observations

It is what it is

Solution

It is morse but when converted it doesn’t give anything good. You need to swap the morse characters

-..- ...- ..-. --... ....- -- .. -. -.. ..--- ....- --... -..- --..

Convert morse to text

XVF74MIND247XZ

Code #2

Observations

You can see the new passcode format but needs a bit of work

Solution

Reverse to get the right passcode format

rbm75loeb952et

Loeb stands for H Richard Loeb. Substitute Loeb for Richard

rbm75richard952et

Code #3

Observations

This is an Ingress Report post, remember the special Ingress Report format ? KEYWORD#XX##XX#

Ni is a valid keyword, no decoding necessary
