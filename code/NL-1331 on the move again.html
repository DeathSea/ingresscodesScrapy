Three codes were hidden in the HTML source of the NL-1331 on the move again post on the Investigate Ingress blog.

[1] e0k3q05022416752d3n4

[2] wph95p602re

[3] en28892klzhvgztire4

Code #1

Observations

20 characters made up of 15 numbers and 5 letters

Solution

We know that the prefix and suffix use 5 letters and 5 numbers. This would leave 10 numbers for the keyword part. Using a skip of 2 you get

ekq52465dn0300217234

Split in two parts

ekq52465dn
0300217234

10 numbers could be an ISBN number. Look it up on Google to find a reference to a book about the Voynich manuscript

ekq52voynich465dn

Code #2

Observations

Passcode format is there but the keyword is only 1 letter. You’ll need manipulate and then find a keyword sub

Solution

The range of the numbers implies that we need to do a modification. It cannot be Atbash as that 0 still doesn’t get in a valid range. Rot 13 gives

jcu62c379er

Now substitute C for see

jcu62see379er

Code #3

Observations

We seem to have 1 extra number than required but you already have the passcode format.

Solution

Those 5 consecutive numbers are probably the prefix and suffix, let’s reverse the string

4eritzgvhzlk29882ne

Split in 2 parts

4eritzgvh
zlk29882ne

The first part doesn’t really give any clue to a possible keyword, let’s try Atbash

4virgates
aop29882mv

A virgate was a unit of land to measure tax assessment. 4 virgates are equal to a hide

aop29hide882mv
