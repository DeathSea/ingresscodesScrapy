Three codes were hidden in the HTML code of the An Enlightened Victory post on the Investigate Ingress blog.

[1] fqb99296gffg69299bqf

[2] inhttyqnarbaneniatzohtiees

[3] 43aotkq46us8

Code #1

Observations

Notice anything repeating in some kind of way?

Hint
If you can spot the passcode format you’ll know the keyword.
Solution

Looks like the string is repeated twice but reversed. Split it into two parts

fqb99296gf fg69299bqf

Now looking at the first part, you can see the passcode format

fqb99 296gf

The fact the the string repeats itself reversed is a mirror effect. Mirror is a valid keyword.

fqb99mirror296gf

Code #2

Observations

All letters, you can see a few numbers in reverse.

Solution

Not sure how to explain that one but here’s how I got it. Seeing those in, ht, ie & es could mean reversed numbers.

Reverse the whole string

seeithoztainenabranqytthni

Split where you see numbers

se ei th oztainenabranqyt th ni

The passcode format is LLL## keyword ###LL. We have 3 numbers at the start so we can assume oz here the letters for the suffix. Same for the prefix, qyt are right before the 2 last numbers.

se ei th oz tainenabran qyt th ni

We now have

783oz tainenabran qyt39

Searching for taineabran on google gives a link to a character of Memory Alpha namen Tain Enabran who was head of the Obsidian order. Obsidian is a valid keyword

qyt39obsidian783oz
